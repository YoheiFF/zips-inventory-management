|-------|-------|---------|------------|---------|------|

- `load-header.js`: ヘッダー動的読み込み・ローディングオーバーレイ・ログアウト処理
- `auth-check.js`: クライアント側認証確認（未認証時 → login.html へリダイレクト）

### SCR-08 商品登録・編集フォーム（/item-form.html）

```
+--------------------------------------------------+
| [共通ヘッダー]                                    |
+--------------------------------------------------+
| 商品登録 / 商品編集                               |
|                                                  |
| 商品名 *: [_________________________]            |
|                                                  |
| 仕入先会社 *: [ドロップダウン ▼]                  |
|                                                  |
| カテゴリ *: [ドロップダウン ▼]                    |
|                                                  |
| 金額（税別）: [_____] 円                          |
|                                                  |
| 在庫数: [_____] 個                               |
|                                                  |
| キーワード: [_________________________]           |
|                                                  |
| 備考: [                              ]           |
|       [                              ]           |
|                                                  |
| 商品画像:                                         |
| +------------------+                            |
| | [クリックして選択] |  ← クリックで file input   |
| |  (プレビュー表示) |                             |
| +------------------+                            |
|  ※ 5MB 以下 / JPEG, PNG, GIF, WEBP              |
|                                                  |
| [← 一覧に戻る]  [削除] (編集時のみ)  [登録/更新] |
+--------------------------------------------------+
```

**要素一覧**:
- テキスト入力: 商品名（必須）、キーワード
- セレクト: 仕入先会社（必須）、カテゴリ（必須）
- 数値入力: 金額、在庫数
- テキストエリア: 備考
- 画像入力: クリックで file input を起動、プレビュー表示
- ボタン: 戻る、削除（編集時のみ表示）、登録/更新（モードに応じて表示切替）
